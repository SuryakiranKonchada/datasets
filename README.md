# datasets

A generic dataset repository for storing reusable data files across projects.

## Purpose
This repository is intended as a centralized data library. It lets you:
- reuse datasets in multiple projects
- keep data files organized
- track dataset history with version control

## How to use
1. Clone this repository when you need shared dataset files.
2. Use dataset files directly from this repository in your projects.
3. Add new datasets as needed.

## Notes
- Keep dataset names descriptive.
- This repository is for data storage and reuse.
